# Launchpad

## Requirements

This project requires Java 11 compiler.


## Usage

Include Launchpad to your project:

```xml
<dependency>
    <groupId>de.focus-shift</groupId>
    <artifactId>launchpad</artifactId>
    <version>...</version>
</dependency>
```
